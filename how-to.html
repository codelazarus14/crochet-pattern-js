How To Use The Generator

Click on the images below to get a full-size view.

To create a pattern, first go to the main page of this site and begin by filling in some basic info. Under the Pattern Options heading, enter a Title, Author and (optionally) a Description for the pattern, before selecting a project type from the dropdown. After confirming your selection the pattern creation form will open underneath.

Alternatively, if you have saved patterns on the site before, you can select a pattern to edit from from the Saved Patterns list. After clicking Load the creation form's fields will be populated from the loaded pattern.

From there, enter the required information in each section of the pattern creation form (Materials, Glossary, Notes etc.).

Steps

When entering Steps, note that each step in the pattern's information is stored separately from their order in the list of steps. This is to allow for flexibility in repeating steps without having to duplicate instructions.

Step indices can be manually edited via the input fields on each step, and rearranged by dragging and dropping steps into the spaces above or below each other. Steps do not change their row indices after being rearranged, so make sure to adjust them afterward!

You can also add Sections to the pattern via the appropriately labeled button. Each section maintains a separate row counter and can be used to split up distinct parts of a larger pattern to be joined together near the end. Beware that deleting a section will also delete all of its contents from the form!

Submit and Use

After completing the pattern creation form and fixing any errors that arise, the pattern will be saved via the Save or Submit buttons. The Save button will save the pattern and allow you to return to creating and editing patterns, while the Submit button allows you to open the pattern preview.

From here you can inspect the output and proceed to either export a PDF or a raw JSON of the saved pattern, or make use of it on the next page.

On the pattern use page, you can choose to load and make use of any saved pattern on the website. The dropdowns to the left allow you to peek at the various details of the pattern as necessary, while the main steps view is displayed to the right. Entering numbers or using the up or down arrows on the Row Counter allows you to progress through the pattern. Pattern progress is saved after refreshing or closing the browser window.
